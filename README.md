# 📒 Консольная книга контактов

## Описание

Данное приложение представляет собой консольную книгу контактов, написанную на языке Python.

Программа позволяет хранить контакты в файле `contacts.txt` и выполнять основные операции по их управлению.

## Возможности программы

- ➕ Добавление нового контакта
- 🔍 Поиск контакта по имени или телефону
- ❌ Удаление контакта
- ✏️ Обновление информации о контакте
- 📋 Просмотр всех контактов с сортировкой по имени
- 💾 Автоматическое сохранение данных в файл `contacts.txt`

## Структура контакта

Каждый контакт содержит:

- Имя
- Телефон
- Email

Пример записи в файле:

```
Иван,+420123456789,ivan@mail.com
```

## Проверка данных

При добавлении и обновлении контакта выполняются проверки:

- имя не должно быть пустым;
- телефон должен состоять только из цифр;
- длина телефона должна быть 12 цифр;
- email должен содержать символы `@` и `.`.

Если введены некорректные данные, программа попросит пользователя повторить ввод.

## Меню программы

После запуска пользователю доступны следующие действия:

```
1. Добавить контакт
2. Найти контакт
3. Удалить контакт
4. Обновить контакт
5. Просмотреть контакты
6. Выйти
```

## Использование

1. Запустите программу:

```bash
python main.py
```

2. Выберите нужный пункт меню.

3. Следуйте инструкциям на экране.

## Используемые функции

В программе реализованы следующие функции:

- `add_contact()` — добавление контакта;
- `find_contact()` — поиск контакта;
- `delete_contact()` — удаление контакта;
- `update_contact()` — обновление контакта;
- `save_contacts()` — сохранение контактов в файл;
- `load_contacts()` — загрузка контактов из файла.
